# NGO Awareness Website

A fully responsive NGO Awareness Website built using React.js

## 🔧 Tech Stack
- React.js
- HTML5, CSS3
- JavaScript

## ✨ Features
- Volunteer registration form
- Donation awareness section
- Campaign highlights page
- Fully responsive design

## 🚀 How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
